Projects

Cottage Management

This solution was a team effort made for the university, the subject was Programming III. It consists of a web app made using .NET Core 7 divided in three smaller projects. The first one is Business Logic, for the second one we have Data Acces in which we used Entity Framework as our ORM, and last but not least is MVC.

Sushi Web

As well as the Cottage Management solution, this was made for the university though not a team effort. The subject was Web Interface Design and final goal was, based on a mockup to design a semantically correct page using HTML, CSS and Bootstrap.

Expense Manager

Unlike my other projects, this wasn't made for the university. It was made for a very specific use case. This solution uses Java as the main language and consists of two projects. The first one is a RESTFul API developed using SpringBoot which connects to a MySQL DB. Second one being the interface, it was made using Java Swing library.

Bookings

Web application made as a university project, it allows to make reservations in different comercial places (without date or hour). The main goal was to implement HTML and Javascript, it doesn't connect with a DB so any data that the user enters will be saved in memory and lost at refresh.
